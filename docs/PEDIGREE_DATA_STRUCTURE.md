# 血統データ構造

## umadataテーブルの血統カラム

新しいumadata.csvには詳細な血統情報が追加されています。

### 既存の血統カラム

| カラム名 | CSV列 | 説明 |
|---------|------|------|
| `sire` | AS (列36) | 種牡馬（父） |
| `dam` | AT (列37) | 母馬 |

### 新規追加された血統カラム

| カラム名 | CSV列 | 説明 |
|---------|------|------|
| `sire_type` | BH (列39) | 父タイプ名 |
| `dam_type` | BI (列40) | 母タイプ名 |
| `broodmare_sire` | BJ (列41) | 母父馬 |
| `broodmare_sire_type` | BK (列42) | 母父タイプ名 |

## タイプ名について

「タイプ名」は種牡馬の系統や特徴を表す分類情報です。

例：
- スピード系
- スタミナ系
- 万能型
- パワー系
など

## 活用方法

### 1. 血統分析の強化

母父の情報を含めることで、より詳細な血統分析が可能になります。

```sql
-- 母父が同じ馬の成績を調べる
SELECT * FROM umadata 
WHERE broodmare_sire = '○○○'
AND distance LIKE '芝2000%';
```

### 2. タイプマッチング

父と母父のタイプの組み合わせから、適性を分析できます。

```sql
-- スピード×スタミナの組み合わせを調べる
SELECT * FROM umadata
WHERE sire_type = 'スピード系'
AND broodmare_sire_type = 'スタミナ系';
```

### 3. 研究AIでの活用

研究AIツールで以下のような分析が可能：
- 母父の影響分析
- 血統タイプの相性分析
- 母系の特徴抽出

## インデックス

血統検索の高速化のため、以下のインデックスが作成されています：

- `idx_umadata_sire` - 父名
- `idx_umadata_dam` - 母名
- `idx_umadata_broodmare_sire` - 母父名
