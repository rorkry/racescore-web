## 📋 今後やるべき分析

### 1. 人気薄での巻き返しパターン
```
【分析内容】
- 6番人気以下で僅差負け → 次走成績
- 人気薄 + 特殊ラップで負け → 次走成績
- 人気薄 + 高指数 → 次走成績（過剰人気にならないか確認）
```

### 2. 特殊ラップの詳細分析
```
【分析内容】
- 各コース・距離別の「出現率が低いラップパターン」を特定
- 特殊ラップで「先行して負けた馬」の次走成績
- 特殊ラップで「後方から差し損ねた馬」の次走成績
- 中弛みラップで「中団にいた馬」の次走成績
```

### 3. 時計評価の詳細分析
```
【分析内容】
- 同条件（コース・距離・馬場）での時計ランキング
- 時計レベル別（トップ10%、上位20%）の次走成績
- ハイレベル時計で負けた馬の巻き返し
```

### 4. 度外視パターンの分析
```
【分析内容】
- 大敗(1秒超) + 高指数 → 次走成績
- 出遅れ・不利明記 + 高指数 → 次走成績
- 人気裏切り + 次走条件変わり → 成績
```

### 5. 複合パターンの深掘り
```
【分析内容】
- レベルA/B + 特殊ラップ + 位置取り + 着差 の組み合わせ
- 「勝者を追いかける」パターンの条件別詳細
- コース替わり・距離変更時の巻き返しパターン
```

## 💡 重要な発見（メモ）

1. **「僅差負け」は儲からない** - みんな狙うので過剰人気
2. **「勝者を追いかける」が儲かる** - 特にレベルC/Bの勝者
3. **ラップ・時計・レベル単体では勝てない** - 指数との組み合わせが必要
